# 🚨 긴급 수정 가이드 - Streamlit Cloud "spawn error" 해결

## 문제 원인
Streamlit Cloud가 Python 3.13을 사용하고 있어 패키지 설치 및 실행에 문제가 발생하고 있습니다.

## 해결 방법 (필수!)

### 1단계: Streamlit Cloud 대시보드 접속
1. https://share.streamlit.io 접속
2. 로그인 후 문제가 있는 앱 선택

### 2단계: Python 버전 변경 (가장 중요!)
1. 앱 대시보드에서 **"Settings"** (⚙️ 아이콘) 클릭
2. **"Advanced settings"** 섹션으로 스크롤
3. **"Python version"** 드롭다운 찾기
4. **"3.11"** 선택 (현재는 3.13으로 되어 있을 것입니다)
5. **"Save"** 버튼 클릭

### 3단계: 재배포 대기
- 저장 후 자동으로 재배포가 시작됩니다
- 약 5-10분 소요됩니다

### 4단계: 로그 확인
재배포 후 로그에서 다음을 확인하세요:
```
Using Python 3.11.x environment
Resolved XXX packages
Installing packages...
```

## 확인 사항

### Python 버전이 변경되었는지 확인
로그에서 다음을 찾으세요:
- ✅ `Python 3.11.x` → 성공
- ❌ `Python 3.13.x` → 아직 변경되지 않음

### 패키지 설치 확인
로그에서 다음을 확인하세요:
- ✅ `Installing packages...` → 정상 진행 중
- ✅ `Successfully installed` → 설치 완료
- ❌ `ERROR` 또는 `ModuleNotFoundError` → 문제 발생

## 추가 문제 해결

### 여전히 오류가 발생하는 경우
1. Streamlit Cloud 로그의 전체 오류 메시지 확인
2. 오류 메시지를 복사하여 공유해주시면 추가 도움을 드리겠습니다

### 패키지 설치가 너무 오래 걸리는 경우
- 정상입니다. 특히 `torch`, `torchvision` 같은 대용량 패키지는 시간이 걸립니다.
- 20분 이상 걸리면 문제일 수 있습니다.

## 참고
- Python 3.11이 가장 안정적이고 모든 패키지와 호환됩니다
- Python 3.13은 아직 일부 패키지가 지원하지 않습니다
- `runtime.txt` 파일은 Streamlit Cloud에서 작동하지 않습니다
